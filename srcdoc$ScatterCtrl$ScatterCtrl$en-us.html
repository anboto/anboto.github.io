ScatterCtrl

 

 

Versatile 2D scatter graph control based on ScatterDraw.

 

It uses U++ Draw or Painter on demand to get better refresh speed or printing quality.

 

ScatterDraw page describes some of the main features, and in ScatterCtrl_demo there is an overview of control possibilities.

 

As a visual control, if allowed by the programmer, the final user can handle some of the features and data:

Data dialog

Process dialog

Properties dialog
