Controls4U_Demo

 

Controls4U_Demo package does a visual review of classes and features in different tabs. Here there are some screenshots of them:

 

 

VLC ActiveX

 

This tab implements a simple but useful player based on VLC ActiveX.

 

 

 

Firefox ActiveX

 

This tab implements a web browser based on Mozilla Firefox ActiveX.

 

 

 

Internet Explorer ActiveX

 

This tab implements a web browser based on Microsoft Internet Explorer ActiveX.

 

 

 

Meter demo

 

Move SliderCtrl in the right to change Meters value and change the properties of the main Meter in the right.

Meter handle is moved using multitasking so it is not affected by window moving or resizing.

 

 

 

StaticClock demo

 

Change StaticClock parameters and resize window to see the control look.

 

 

 

StaticImage and EditFile demo. StaticImage appears as horses picture and as paper like background

 

Choose the image file to be opened and change some of its properties.

background pattern is also an StaticImage.

 

 

 

StaticLine, StaticArrow, StaticRectangle and StaticEllipse demo in a flowchart

 

Some Bazaar packages and their organization.

 

 

 

Test card for StaticLine, StaticArrow, StaticRectangle and StaticEllipse

 

Just a test to see how static controls fit between themselves.
